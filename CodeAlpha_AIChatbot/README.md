# AI Chatbot

## Description
A simple rule-based AI Chatbot developed using Java for the CodeAlpha Internship. The chatbot responds to predefined user queries.

## Features
- Responds to greetings
- Answers basic questions
- Interactive chat interface
- Exit using "bye"

## Technologies Used
- Java
- Scanner Class
- Conditional Statements

## How to Run
javac AIChatbot.java
java AIChatbot
